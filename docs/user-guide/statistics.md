# Statistics

Track your productivity and gain insights into your work patterns with FlickDo's statistics features.

## Overview

Access statistics by clicking **"Statistics"** in the sidebar. The dashboard shows:

- Completion metrics
- Task distribution
- Productivity trends
- Time analytics

## Key Metrics

### Completion Rate

**Overall Completion Rate**

- Percentage of tasks completed vs created
- Calculated over different time periods
- Color-coded: 🟢 >80%, 🟡 60-80%, 🔴 <60%

**Daily Completion**

- Tasks completed today
- Comparison to your average
- Streak tracking

**Weekly Summary**

- Tasks completed this week
- Week-over-week comparison
- Best day of the week

## Privacy

Your statistics are:

- ✅ Private by default
- ✅ Never shared without permission
- ✅ Can be opted out of team stats
- ✅ Exportable and deletable

## Troubleshooting

### Stats Not Updating

- Refresh the page
- Stats update hourly
- Check date range filter

### Missing Tasks in Stats

- Ensure tasks aren't filtered out
- Check if list is archived
- Verify date range includes task dates

### Incorrect Calculations

- Stats recalculate overnight
- Force recalculation: Settings → Advanced → Recalculate Stats
- Report bugs on GitHub
