## Managing Lists

### Edit a List

1. Click the list name in sidebar
2. Click the settings icon (âš™ï¸)
3. Select "Edit List"
4. Update properties
5. Save changes

### Reorder Lists

Drag and drop lists in the sidebar to arrange them in your preferred order.

### Archive a List

Completed a project? Archive it:

1. Open list settings
2. Select "Archive"
3. Tasks remain accessible in archive

Archived lists:

- Don't appear in main sidebar
- Can be restored anytime
- Tasks don't show in global views

### Delete a List

!!! danger "Permanent Action"
Deleting a list also deletes all its tasks. This cannot be undone!

1. Open list settings
2. Select "Delete List"
3. Confirm deletion

Consider archiving instead of deleting.

## Sharing & Collaboration

### Share a List

1. Open the list

### Permission Levels

**Viewer**

- See all tasks
- Cannot edit anything

**Editor**

- Create and edit tasks
- Cannot manage members or list settings

**Admin**

- Full access
- Can invite/remove members
- Can edit list settings

## Best Practices

!!! tip "Keep Lists Focused"
5-15 lists is ideal. Too many becomes overwhelming.

!!! tip "Use Consistent Naming"
Develop a naming convention: "Project: Name" or "Client - Project"

!!! tip "Color Code Consistently"
Use the same colors for similar types of lists across your workspace.

!!! tip "Regular Cleanup"
Archive completed projects monthly to keep your workspace tidy.

!!! tip "Share Appropriately"
Only share lists that truly need collaboration. Keep personal tasks private.

## Troubleshooting

**Can't share a list?**

- Check that you have admin permissions
- Verify the email address is correct
- Ensure the user has a FlickDo account

**List not syncing?**

- Refresh the page
- Check internet connection
- Try logging out and back in

**Wrong list order?**

- Drag and drop to reorder
- Order is saved automatically
