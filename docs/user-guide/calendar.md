# Calendar

The calendar view provides a visual way to plan and manage your tasks over time.

## Overview

Access the calendar by clicking **"Calendar"** in the sidebar. It shows:

- Tasks with due dates
- Google Classroom assignments
- Events and deadlines
- Color-coded by list

## Views

### Month View

See an overview of the entire month:

- All tasks displayed on their due dates
- Multiple tasks per day shown as a list
- Click any day to see details
- Navigate with arrows or date picker

### Week View

Detailed weekly planning:

- 7-day view with time slots
- Hourly breakdown for time-specific tasks
- Drag and drop to reschedule
- Better for detailed planning

### Day View

Focus on a single day:

- Hour-by-hour schedule
- All tasks for the selected day
- Time blocking view
- Best for daily planning

## Creating Tasks from Calendar

### Quick Create

1. Click any date in the calendar
2. Enter task title
3. Press `Enter`

Task is created with that due date.

### Time-Specific Tasks

1. Click a time slot in week/day view
2. Enter task details
3. Due date and time are pre-filled

## Managing Tasks in Calendar

### Reschedule by Drag & Drop

1. Click and hold a task
2. Drag to a new date/time
3. Release to update

### Edit Task

Click any task to open the full details and edit.

### Complete Task

Click the checkbox on the task card.

## Filtering

### By List

- Select lists from the filter dropdown
- Show/hide specific lists
- Toggle all lists on/off

### By Tag

- Filter by one or more tags
- Useful for seeing specific types of tasks

### By Status

- Show only incomplete tasks
- Include or exclude completed tasks

## Google Classroom Integration

Classroom assignments appear automatically:

- Labeled with course name
- Different visual style
- Click to open in Google Classroom
- Updates sync automatically

## Color Coding

Tasks are colored by their list:

- Quickly identify which project/category
- Customize list colors in settings
- Consistent with main task view

## Navigation

- **Today button** - Jump to current date
- **Arrow buttons** - Previous/next period
- **Date picker** - Jump to any date
- **View selector** - Switch between month/week/day

## Keyboard Shortcuts

- `t` - Go to today
- `←` / `→` - Previous/next period
- `m` - Month view
- `w` - Week view
- `d` - Day view
- `n` - New task on selected date

## Best Practices

!!! tip "Time Blocking"
Use the week view to time-block your day. Assign specific hours to tasks.

!!! tip "Weekly Review"
Every Monday, review the week ahead in week view. Adjust priorities.

!!! tip "Color Coordination"
Use list colors strategically. For example: red for urgent projects, blue for ongoing work.

!!! tip "Daily Planning"
Each morning, check day view and prioritize your tasks for the day.

## Tips & Tricks

### Recurring Events

While recurring tasks aren't built-in yet, you can:

1. Create a task template
2. Duplicate it weekly/monthly
3. Adjust dates as needed

### All-Day Events

Tasks without specific times appear at the top of each day.

### Overlapping Tasks

If multiple tasks have the same time:

- They stack vertically
- Scroll to see all
- Consider rescheduling to avoid conflicts
