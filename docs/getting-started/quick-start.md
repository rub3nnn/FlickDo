# Quick Start

Get up and running with FlickDo in just a few minutes! This guide assumes you've already completed the [installation](installation.md).

## First Login

1. Navigate to `http://localhost:5173`
2. Click **"Sign Up"**
3. Choose one of these options:
   - Sign up with email and password
   - Sign in with Google

## Create Your First Task

### Method 1: Quick Add (Recommended)

1. Press `Ctrl+K` (or `Cmd+K` on Mac) to open the command palette
2. Type your task name
3. Press `Enter`

That's it! Your task is created.

### Method 2: Full Form

1. Click the **"+ New Task"** button in the sidebar
2. Fill in the details:
   - **Title**: What needs to be done?
   - **Description**: Additional context (optional)
   - **Due Date**: When is it due?
   - **Priority**: Low, Medium, or High
   - **Tags**: Add relevant tags (optional)
3. Click **"Create Task"**

## Organize with Lists

### Create a List

1. In the sidebar, click **"+ New List"**
2. Enter a name (e.g., "Work Projects")
3. Choose a color and icon
4. Click **"Create"**

### Add Tasks to Lists

When creating a task, select the list from the dropdown, or drag-and-drop existing tasks into lists.

## Use Tags

### Create a Tag

1. Open any task
2. Click in the **Tags** field
3. Type a new tag name (e.g., "urgent")
4. Press `Enter`

### Filter by Tags

Click any tag in the left sidebar to see all tasks with that tag.

## Essential Keyboard Shortcuts

- `Ctrl+K` - Open command palette
- `Ctrl+N` - New task
- `Ctrl+L` - New list
- `/` - Focus search
- `Esc` - Close dialogs

## Google Classroom Integration

### Connect Your Account

1. Click your profile in the top-right
2. Select **"Settings"**
3. Go to **"Integrations"** tab
4. Click **"Connect Google Classroom"**
5. Authorize the app

Your assignments will automatically sync as tasks!

## View Your Calendar

1. Click **"Calendar"** in the sidebar
2. See all your tasks on a calendar view
3. Drag tasks to reschedule them

## Check Your Stats

1. Click **"Statistics"** in the sidebar
2. View:
   - Tasks completed today/this week
   - Completion rate
   - Tasks by priority
   - Activity chart

## Share a List

1. Open any list
2. Click the share icon (⚙️ → Share)
3. Enter a collaborator's email
4. Choose their permission level:
   - **View**: Can see tasks
   - **Edit**: Can modify tasks
   - **Admin**: Can manage members
5. Click **"Invite"**

## Pro Tips

!!! tip "Quick Navigation"
Use `Ctrl+K` to quickly jump to any list, task, or feature. It's the fastest way to navigate FlickDo!

!!! tip "Batch Actions"
Select multiple tasks with checkboxes to complete, delete, or move them all at once.

!!! tip "Smart Filters"
Use the filter tabs (All, Today, Upcoming, Completed) to focus on what matters now.

## Common Workflows

### Daily Planning

1. Check **"Today"** view each morning
2. Drag high-priority tasks to the top
3. Mark tasks complete as you finish them

### Project Management

1. Create a list for each project
2. Use tags for task types (design, development, review)
3. Assign tasks to team members
4. Track progress in Statistics

### Student Workflow

1. Connect Google Classroom
2. Use tags for each subject
3. View assignments in Calendar
4. Check Today view before class
